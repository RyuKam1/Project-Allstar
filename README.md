## üõ†Ô∏è Tech Stack & Architecture

This project is built with a focus on performance, scalability, and premium UX.

### Core Stack
*   **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
*   **UI Library**: [React 19](https://react.dev/)
*   **Language**: JavaScript (ES6+) / Node.js

### Styling & Design
*   **CSS Modules**: Modular, scoped styling for maintainable components.
*   **Vibe Engine**: A bespoke design philosophy prioritizing visual excellence, micro-animations, and fluid responsiveness without relying on heavy UI frameworks.

### Backend & Data
*   **Supabase**:
    *   **Auth**: Secure user authentication and session management.
    *   **Database**: PostgreSQL for robust relational data storage.
    *   **Storage**: Optimize image hosting for user uploads.

### Integrations
*   **Leaflet / React-Leaflet**: Open-source interactive maps.
*   **Date-fns**: Comprehensive date manipulation.
*   **Supabase SSR**: Server-side rendering support for authentication.

## üèÅ Getting Started

To run this project locally:

1.  **Clone the repository**
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Set up Environment Variables**:
    Create a `.env.local` file with your Supabase credentials.
4.  **Run the development server**:
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) to see the application in action.
